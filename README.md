# HPDB

- Users can creat accounts, add headphones to the collection, post listings, and post reviews.
- User accounts are authenticated using JWT and cookies.

## Programming languages and versions

### Languages

- Typescript

### Frameworks

- Express.js
- Prisma
- GraphQL-Yoga
- AWS S3

### Database Systems

- CockroachDB
- Tebi Cloud

### Hosting

- API hosted on fly.io
